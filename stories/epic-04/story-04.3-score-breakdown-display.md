# Story 04.3: Score Breakdown Display

## User Story
**As a** recruiter  
**I want to** see a detailed breakdown of how a candidate's fit score was calculated  
**So that** I understand which criteria they meet and where they fall short

## BRD Requirements Covered
- FR-SC-03: Score breakdown shows contribution of each criterion (skills, experience, education)
- BRD Section 8.8: Explainability Layer — fit score breakdown via weighted feature attribution

## Acceptance Criteria
1. **Given** a fit score is displayed on a candidate card  
   **When** the recruiter clicks "Score Breakdown" or "Why this score?"  
   **Then** a panel opens showing each scoring dimension with its weight, raw similarity, and point contribution

2. **Given** the breakdown panel is open  
   **When** the recruiter views the Technical Skills dimension  
   **Then** they see: matched skills (green), missing skills (red), and inferred skills (amber)

3. **Given** the breakdown panel is open  
   **When** the recruiter views the Experience dimension  
   **Then** they see a plain-English explanation of how the candidate's experience aligns with the JD

4. **Given** the breakdown is displayed  
   **When** the recruiter hovers over a criterion bar  
   **Then** a tooltip shows the specific evidence from the candidate's profile

5. **Given** the breakdown panel is open  
   **When** the recruiter clicks "Export Explanation"  
   **Then** a PDF is generated with the full score breakdown and evidence

## Priority
**P0 — Must Have**

## Estimated Effort
**5 story points**

## NFR / Tech Notes
- **Panel Load:** ≤ 500ms (data already computed at scoring time)
- **Evidence Text:** Generated by LLM at scoring time and cached
- **PDF Export:** Generated server-side using Puppeteer or WeasyPrint

## Technical Design

### Breakdown Panel Component
```typescript
interface ScoreBreakdown {
  overall: number;
  dimensions: {
    name: string;
    weight: number;
    score: number;
    contribution: number;
    matched_items: string[];
    missing_items: string[];
    evidence: string;
  }[];
}
```

### API Endpoints
```
GET  /api/applications/:id/score/breakdown   — Get full breakdown with evidence
POST /api/applications/:id/score/export-pdf  — Export breakdown as PDF
```
